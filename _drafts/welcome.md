# Overview

Alien Base is committed to elevating the DeFi trading experience with a user-friendly, CEX-inspired interface while maintaining the privacy and control benefits unique to decentralized finance.

Through its innovative Epsilon technology, Alien Base seamlessly aggregates all pools on the Base network, selecting the safest and most efficient pathways for quicker and more cost-effective swaps.

Additionally, with features like our token generator and straightforward pool deployment, Alien Base simplifies the trading process for users and enables developers to launch their own tokens in mere seconds.

This integration of advanced technology ensures a superior trading experience that is both user-friendly and developer-friendly.

## Mission Statement

Alien Base exists to elevate DeFi by combining its open, permissionless potential with the intuitive user experience found in top centralized exchanges. Our platform empowers both seasoned and new DeFi users by providing robust, accessible trading and liquidity tools that simplify and enhance the DeFi experience.

Our approach combines the best existing technologies with innovative new solutions, all wrapped in a streamlined, user-friendly design to serve traders and builders alike. As a comprehensive v2 and v3 DEX, we deliver efficient swaps, robust liquidity pools, and incentive programs while maintaining complete protocol agnosticism — aggregating liquidity across Base to support every token and pool.

By making DeFi simpler, more engaging, and packed with powerful functionality, Alien Base aims to increase adoption across the ecosystem, not only for basic features like token swaps but also for the diverse, creative opportunities that DeFi unlocks.

## Alien Base Technology

Alien Base is a Meta-DEX native to Base, merging multiple existing technologies to offer a comprehensive DeFi trading experience:

- **Alien Base v2** — Built on Uniswap v2 contracts, this was Alien Base's original DEX technology launched in August 2023, offering a straightforward and reliable trading interface.
- **Alien Base Farming Program** — Derived from PancakeSwap's contracts and interface, Alien Base launched with a fair liquidity incentivization program using emissions of its native token, $ALB.
- **Alien Base v3** — Leveraging Uniswap v3's concentrated liquidity model, Alien Base fully supports concentrated liquidity technology, allowing more efficient use of capital.
- **Alien Base Bunni** — A Uniswap v3 wrapper based on Bunni Protocol, this feature transforms Uniswap positions into ERC-20 tokens, adding flexibility and utility over NFT-based positions.
- **ERC-20 Locking system** — Based on Camelot's xGrail model, this system allows any ERC-20 token, including LP tokens, to be locked, enabling creations like esALB and esLP (locked ALB-ETH farm positions).
- **Epsilon** — Alien Base's proprietary meta-aggregator, optimizes trades by sourcing the best rates from major DEXs, aggregators, and Alien Base's own pools.
- **Token Generator** — Part of a growing suite for developers (called "Area51"), this tool allows secure, rapid token deployment on Base.
- **Alien Base Carbon** — Powered by Bancor's Carbon technology, this feature introduces Limit Orders and Automated Strategies, expanding Alien Base's trading capabilities.
